# Public cloud service review

Public cloud services should be tied to approved architecture decisions,
centralized logging, least-privilege access, and documented data sensitivity.
Unexpected public exposure should be validated with the service owner.

# Defensive controls

Prefer managed access controls, network segmentation, monitored administrative
access, and change records that explain why a service is reachable.
